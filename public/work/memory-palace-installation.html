Memory Palace Installation

2022 Installation, AR, Interactive Completed

An interactive installation that visualizes collective memory through spatial computing and augmented reality interfaces.

About This Work

Memory Palace is an immersive installation that explores collective memory and spatial cognition. Visitors navigate through a physical space augmented with digital overlays that respond to their movement and presence. The piece draws from the ancient method of loci, where memories are associated with specific locations. Using spatial computing technologies, the installation creates a shared memory space where individual experiences contribute to a collective narrative. Motion sensors detect visitor movement, triggering audiovisual responses that build upon previous interactions. The work questions how digital technologies can enhance or distort our relationship with memory and place.
